# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Commands

```bash
npm run setup        # First-time setup: install deps, generate Prisma client, run migrations
npm run dev          # Start dev server (Turbopack) at http://localhost:3000
npm run build        # Production build
npm run lint         # ESLint
npm run test         # Run all Vitest tests
npm run db:reset     # Reset SQLite database (destructive)
```

Run a single test file:
```bash
npx vitest src/components/chat/__tests__/ChatInterface.test.tsx
```

## Architecture

UIGen is a Next.js 15 app that lets users describe React components in chat, then shows live previews generated by Claude AI.

### Three-Panel Layout
The main UI (`src/app/main-content.tsx`) is a resizable 3-panel layout: Chat (35%) | Preview/Code editor (65%). State lives in two React contexts:
- `src/lib/contexts/chat-context.tsx` — chat messages via `@ai-sdk/react` `useChat` hook
- `src/lib/contexts/file-system-context.tsx` — virtual file system state

### Virtual File System
`src/lib/file-system.ts` implements an in-memory `VirtualFileSystem` class — no disk writes. Files serialize to/from JSON strings for persistence in Prisma. The AI writes files here via tool calls.

### AI Integration & Tool Calls
`src/app/api/chat/route.ts` streams responses from Claude (claude-haiku-4-5 by default) with two tools:
- `src/lib/tools/str-replace.ts` — text editor tool (view/create/str_replace/insert operations)
- `src/lib/tools/file-manager.ts` — rename/delete operations

The system prompt (`src/lib/prompts/generation.tsx`) instructs Claude to use `/App.jsx` as the entry point and `@/` prefix for inter-file imports.

If `ANTHROPIC_API_KEY` is missing from `.env`, `src/lib/provider.ts` falls back to a mock provider with static responses (useful for offline dev).

### Code Preview Pipeline
`src/lib/transform/jsx-transformer.ts` uses `@babel/standalone` to transform JSX/TSX at runtime, maps `@/` imports to an esm.sh CDN import map, and injects the result into an iframe (`src/components/preview/PreviewFrame.tsx`).

### Authentication
JWT sessions via `jose` with HTTP-only cookies (7-day expiry). `JWT_SECRET` defaults to `"development-secret-key"` if not set. Auth middleware protects `/api/projects` and `/api/filesystem` routes.

### Database
Prisma with SQLite (`prisma/dev.db`). The full schema is in `prisma/schema.prisma` — reference it whenever you need to understand the structure of data stored in the database. Authenticated users get project persistence; anonymous users get session-only state.

### Path Alias
`@/*` maps to `./src/*` throughout the codebase.
